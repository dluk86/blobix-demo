English Version

This Privacy Policy describes how the demo version of the mobile game - Blobix Demo ("Application") handles user data.

1. General Information

This is a demo version of the application, provided free of charge and without advertisements.

2. Data Collection

The Application does not collect, store, or process any personal data.

The Application does not require user registration and does not request any personal information.

3. Analytics and Tracking

The Application does not use any analytics tools, tracking technologies, or third-party SDKs for data collection.

4. Advertising

The Application does not display advertisements and does not use advertising SDKs.

5. Permissions

The Application does not request access to sensitive device permissions such as location, contacts, camera, or microphone.

6. Children's Privacy

The Application does not knowingly collect any personal data from users, including children.

7. Data Sharing

No user data is collected, therefore no data is shared with third parties.

8. Changes to This Policy

This Privacy Policy may be updated in the future. Any changes will be published on this page.

Wersja polska

Blobix Demo - polityka prywatności

Niniejsza Polityka prywatności opisuje sposób przetwarzania danych w wersji demonstracyjnej gry mobilnej Blobix Demo („Aplikacja”).

1. Informacje ogólne

Jest to wersja demonstracyjna aplikacji, udostępniana bezpłatnie i bez reklam.

2. Zbieranie danych

Aplikacja nie zbiera, nie przechowuje ani nie przetwarza danych osobowych.

Aplikacja nie wymaga rejestracji ani podawania jakichkolwiek danych osobowych.

3. Analityka i śledzenie

Aplikacja nie korzysta z narzędzi analitycznych, technologii śledzenia ani zewnętrznych SDK zbierających dane.

4. Reklamy

Aplikacja nie wyświetla reklam i nie korzysta z żadnych sieci reklamowych.

5. Uprawnienia

Aplikacja nie wymaga dostępu do wrażliwych uprawnień urządzenia, takich jak lokalizacja, kontakty, aparat czy mikrofon.

6. Prywatność dzieci

Aplikacja nie zbiera żadnych danych osobowych użytkowników, w tym dzieci.

7. Udostępnianie danych

Aplikacja nie zbiera danych, w związku z czym nie udostępnia ich żadnym podmiotom trzecim.

8. Zmiany w polityce prywatności

Polityka prywatności może zostać zaktualizowana w przyszłości. Wszelkie zmiany będą publikowane na tej stronie.
